```python
def je_prost(broj):
    """
    Proverava da li je broj prost.
    
    Args:
        broj (int): Broj koji se proverava za prostost.
    
    Returns:
        bool: Da li je broj prost.
    """
    if broj <= 1:
        # Broj 1 i 0 nisu prosti
        return False
    
    for i in range(2, int(broj ** 0.5) + 1):
        # Proverava da li je broj deljiv s nekim brojem manjim od njega
        if broj % i == 0:
            # Broj je prost samo ako nije deljiv na neki broj manjim od njega
            return False
    
    # Broj je prost ako nije deljiv na neki broj manjim od njega
    return True

# Primjer upotrebe

    print(broj, "nije prosti broj.")
```

3. Broj je prost samo ako nije deljiv na neki broj manjim od njega.
