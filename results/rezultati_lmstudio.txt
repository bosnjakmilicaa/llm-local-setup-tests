```python
def je_prost(broj):
    """
    Proverava da li je broj prost.

    :param broj: Int, broj koji se proverava
    :return: bool, True ako je broj prost, False inače
    """
    if broj < 2:
        return False

    for i in range(2, int(broj ** 0.5) + 1):
        if broj % i == 0:
            return False

    return True

# Primjer:
print(je_prost(7))  # Output: True
print(je_prost(4))  # Output: False
```

Objasnjenje koda:

1. Funkcija `je_prost` prima jedan parametar `broj`.
2. Proverava se da li je broj manji od 2, što nije prost.

4. Ukoliko je proverljena deljivost broja sa npr. 2, funkcija vrati `False`.
5. Ako za sve iteracije petlje nije pronadjen broj koji deli broj, funkcija vrati `True`.

```python
def is_prime(number):
  """
  This function checks if a number is prime.

  Args:
    number: The integer to check for primality.

  Returns:
    True if the number is prime, False otherwise.
  """
  # Handle special cases
  if number <= 1:
      return False
  if number <= 3:
      return True

  # Check divisibility by 2 and 3
  if number % 2 == 0 or number % 3 == 0:
      return False

  i = 5
  while i * i <= number:
    if number % i == 0 or number % (i + 2) == 0:
      return False
    i += 6

  return True

# Example usage

else:
  print(f"{number} is not a prime number.")
```

   - The loop continues as long as `i * i <= number`. This ensures we check only up to the square root of the number.
   - Inside the loop, we check divisibility by `i` and `(i + 2)`. If a divisor is found, the number is not prime (return `False`).

```python
def je_prost(broj):
    """
    Proverava da li je broj prost.

    :param broj: Broj koji se treba proveriti
    :return: Ili ne, broj je prost ili ne.
    """
    if broj < 2:
        return False
    for i in range(2, int(broj ** 0.5) + 1):
        if broj % i == 0:
            return False
    return True

# Primer upotrebe:
print(je_prost(11))   # Ili ne, 11 je prost ili ne.
print(je_prost(100))  # Ili ne, 100 nije prost ili ne.
```

Izvršena logika koja se koristi u ovoj funkciji, se baza na sledeći princip:

*   ako broj manji ili jednak 2, on nije prost (pača ili jedan);
*   ako broj je prost, mora imati samo 1 i sebe kao faktore;
*   prethodno odredene mogu se izbaciti ciklus jer ako nema nijednu različitog delitelja, onda je prost.
